# 训练与调试记录规则

这个目录只放“过程记录”和“测试证据”。

## 放置规则

- `01_训练日志/`
  - 今天练了什么
  - 推进到哪一步
  - 下一次先做什么
- `02_环境安装记录/`
  - 软件安装
  - 驱动配置
  - VS Code / Keil / CubeMX / Programmer / 串口工具状态
- `03_问题排查/`
  - Multisim 异常
  - 下载失败
  - 串口异常
  - 编译配置问题
- `04_联调记录/`
  - 板级联调
  - 模块联调
- `05_测试证据/`
  - 截图
  - 串口日志
  - 波形
  - 视频
- `99_历史归档/`
  - 不再维护但需要保留的旧记录

## 不放什么

- 不放长期导航说明
- 不放训练计划
- 不放讲义原件

这些内容统一放到：

- `00_开始这里/`

## 当前约定

从现在开始：

1. 日志写进 `01_训练日志/`
2. 环境变化写进 `02_环境安装记录/`
3. 故障排查写进 `03_问题排查/`
