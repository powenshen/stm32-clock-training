# 电子钟需求拆解

这份文档只针对你当前的训练主线，不追求一开始做成豪华版。

## 1. 基础版目标

1. 时间走时
2. 时间显示
3. 按键修改时间

这是你近期必须拿下的最小闭环。

## 2. 标准版目标

1. 闹钟
2. 蜂鸣器提示
3. 10 秒无操作返回
4. 参数保存

## 3. 进阶版目标

1. 红外遥控
2. 长按连加连减
3. 界面优化
4. 静音或显示开关

## 4. 软件位视角的模块拆解

### 输入

1. 按键
2. 红外接收（进阶）

### 处理

1. 系统节拍
2. 菜单状态机
3. 时间逻辑
4. 闹钟逻辑
5. 参数保存逻辑

### 输出

1. LCD 或数码管显示
2. 蜂鸣器
3. 串口调试日志

## 5. 开发顺序

1. 工程骨架
2. LED 与按键
3. 1ms 节拍
4. 串口调试
5. 显示
6. RTC
7. 菜单与改时
8. 闹钟
9. 参数保存
