# STM32 Clock Training

本仓库用于管理电子钟训练相关的需求、方案与代码，当前主工程位于：

- `03_软件工程/电子钟训练工程/`

## 分支约定

- `main`：不停被魔改的内容。
- `study/recomment`：基础版本，没有实际意义。
- `feat/software-complete`：理论上实现，实际上还未调试。
- `feat/hardware-integrated`：未来的完整版会放在这里。
- `test/non-hw-debug`：正在进行软件调试的版本。

## 目录

- `01_需求与题目分析/`：题目要求、需求拆解。
- `02_方案设计/`：软件结构、状态机、引脚分配、调试策略。
- `03_软件工程/`：电子钟训练工程和后续主工程。
- `04_硬件接口资料/`：STM32指南者开发板原理图、引脚对应关系、接线说明。
