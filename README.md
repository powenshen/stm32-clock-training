# STM32 电子钟训练工程

基于 STM32F103（野火指南者开发板）的电子钟项目，适用于电子设计竞赛培训。

**主工程：** `03_软件工程/电子钟训练工程/`

**开发环境：** Keil MDK + VSCode (Keil Assistant)

## 分支

| 分支 | 说明 |
|------|------|
| `main` | 主分支，持续迭代中 |
| `test/non-hw-debug` | 纯软件调试（当前活跃） |
| `feat/software-complete` | 软件功能完整版 |
| `feat/hardware-integrated` | 软硬件联调版本 |
| `study/recomment` | 基础参考版本 |

## 目录结构

| 目录 | 内容 |
|------|------|
| `01_需求与题目分析/` | 题目要求与需求拆解 |
| `02_方案设计/` | 软件架构、状态机、引脚分配、调试策略 |
| `03_软件工程/` | 电子钟训练工程与后续主工程 |
| `04_硬件接口资料/` | STM32 指南者开发板原理图、引脚对照、接线说明 |
